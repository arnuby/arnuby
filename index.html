GIT Y GITHUB

Git es un sistema de control de versiones distribuido. Git fue impulsado por Linus Torvalds y el equipo de desarrollo del kernel de linux. Eellos estaban usando otro sistema de control de versiones de codigo abierto, que ya por aquel entonces era distribuido . todo iba bien hasta que los gestores de aquel sistema de control de versiones lo convirtieron en un software propietario.

GITHUB es un servicio para alojamiento de repositorios de software gestionados por el sistema de control de versiones Git. por tanto Git es algo mas general que nos sirve para controlar el estado de un desarrollo a lo largo del tiempo, mientras Github es algo mas particular, es utilizado en un sitio web pensando para hacer posible compartir el codigo de manera mas facil y al mismo tiempo darle popularidad a la herramienta de control de versiones en si, que es Git.
